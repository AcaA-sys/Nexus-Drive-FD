🛡️ Nexus-Drive-FD V3.0 — Titanium Edition


Nexus-Drive-FD — это бескомпромиссный промышленный контроллер бесщеточных моторов (BLDC/PMSM). В основе проекта лежит принцип абсолютной надежности, прецизионной метрологии и защиты авторских прав.

💎 Ключевые особенности (Industrial Grade)

⚡ Силовая мощь и Защита
- Силовая шина: 24В, ток до 20А.
- Smart Guard: Интеллектуальная защита на BTS660P (софт-старт, защита от КЗ и переполюсовки).
- Active Brake: Выделенный канал тормозного резистора на NTMFS5C628 с аппаратной диагностикой целостности цепи.
- Thermal Control: Мониторинг температуры ключей через DS18B20 и управление охлаждением (PWM Fan).

🎯 Метрология и Точность
- Sin/Cos Engine: Поддержка  оптических/магнитных линеек с аналоговым выходом.
- Analog Front-End: Инструментальные усилители AD8226 с двухполярным питанием.
- Precision Ref: Опорное напряжение 1.65V на базе REF2033 для идеальной центровки АЦП.
- Math Power: Использование аппаратного модуля CORDIC в STM32G4 для мгновенного вычисления угла.

🔄 Универсальный контур обратной связи (Feedback)
- High-Speed Encoder Support: Полная поддержка энкодеров стандарта A, B, Z, ER (Error).
- Differential Signal Processing: Все сигналы энкодера обрабатываются в дифференциальном виде через быстродействующие приемники (AM26LV32), что исключает влияние электромагнитных наводок от силовых кабелей на дистанциях до 5-10 метров.
- Hall Sensors Integration: Предусмотрены входы для датчиков Холла двигателя для уверенного старта и работы FOC-алгоритмов на низких оборотах.
- Multi-Standard Compatibility: Возможность работы как с классическими цифровыми энкодерами, так и с Sin/Cos линейками (через инструментальный тракт AD8226).

🛡️ Гальваническая развязка и периферия
- Isolated I2C Bus: Реализована на базе цифрового изолятора ADuM1250 с выделенным питанием от DC/DC преобразователя B0505S-1W. Это гарантирует полную защиту процессора от помех и разности потенциалов при подключении внешних датчиков.

🔍 Система мониторинга и виброанализа
- Active Resonance Control: Поддержка внешних акселерометров (напр. MPU6050) по изолированной шине I2C для контроля резонансов станины.
- Black Box Logging: Возможность записи спектра вибраций в скоростную память W25Q128 для последующего анализа «здоровья» механики и подшипников.
- Safety Interlocks: Программные пороги по ускорению для мгновенной остановки системы при механических столкновениях или аномальных вибрациях.

🔊 Диагностика и Интерфейс
- Voice Diag: Выход для внешнего УНЧ для голосового оповещения о статусе системы.
- Sync Port: Универсальный выход PA4 (DAC) и пин синхронизации C14 для работы с осциллографом.
- Isolated Bus: Полная гальваническая развязка логики и внешней периферии.
- Diag UART.

Sensors: INA240 (Current), TMP117 (Precise Temp).
Memory: Внешняя F-RAM (опционально для логов).

📍 Nexus-Drive-FD Pinout Map (STM32G431RBT6)

1. Метрология и Энкодер (Sin/Cos & Digital)
Пин	Функция	Описание
- PB12	ADC1_IN11	Аналоговый вход SIN (после AD8226)
- PB1	ADC1_IN12	Аналоговый вход COS (после AD8226)
- PB11	ADC1_IN14	Мониторинг опорного напряжения 1.65V (VREF)
- PB2	TIM2_CH4	Цифровой энкодер: Канал A (через AM26LV32)
- PB3	TIM2_CH2	Цифровой энкодер: Канал B (через AM26LV32)
- PA0	TIM2_CH1	Цифровой энкодер: Канал Z (Index)
- PA15: GPIO_In / EXTI — Канал ER (Error энкодера).
- Halls	(Hall Sensor Interface)
- PC6 / A
- PC7 / C 
- PC8	/ B 

2. Силовое управление (Motor Control)
Пин	Функция	Описание
- PA8,9,10	TIM1_CH1,2,3	PWM выходы на затворы (High Side)
- PB13,14,15	TIM1_CH1,2,3N	PWM выходы на затворы (Low Side)
- PA6	ADC2_IN3	Мониторинг цепи тормозного резистора (Brake Check)
- PA5	GPIO_Out	Управление тормозным ключом (Brake FET)

3. Датчики
Пин	Функция	Описание
- PA1	ADC1_IN2	Мониторинг напряжения шины питания (VBUS)
- PA2	ADC1_IN3	Датчик тока фазы A (INA240)
- PA3	ADC1_IN4  	Датчик тока фазы B (INA240)
- PB0	ADC1_IN15	Датчик тока фазы C (INA240)
- PC0	1-Wire	Шина данных для DS18B20 
- PA7	TIM3_CH2	Управление вентилятором охлаждения (Fan PWM)

4. Сервис, Звук
Пин	Функция	Описание
- PA4	DAC1_OUT1	Аналоговый выход (Осциллограф) / Звук на УНЧ
- C14	GPIO_Out	Синхроимпульс (Sync Out) / Управление Mute УНЧ
- C13 Выбор режима / Кнопка

5. Коммуникация
Пин	Функция	Описание
- PA11	CAN_RX	Изолированный интерфейс CAN-FD
- PA12	CAN_TX	Изолированный интерфейс CAN-FD

6. Board ID
- Board ID (Bit 0)	PC3	In/Out	Адрес (LSB) + Индикация (напр. Status)
- Board ID (Bit 1)	PC4	In/Out	Адрес + Индикация (напр. CAN Activity)
- Board ID (Bit 2)	PC5	In/Out	Адрес (MSB) + Индикация (напр. Error/Brake)

7. I2C
- I2C1 (Internal)	PB6 (SCL), PB7 (SDA)	EEPROM AT24C32D
- EEPROM WP	PC15	GPIO_Out	Защита записи калибровочных данных.
- I2C1: Изолированная шина ADuM1250 (PB6/PB7) —  Датчик вибрации MPU6050 + Термометр станины TMP117.
- PC10/11	UART3_TX/RX	Сервисный диагностический порт
